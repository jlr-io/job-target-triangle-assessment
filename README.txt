Job Target - Technical Assessment | Justin Inho

Requirments:
Starting at the top of the triangle and moving to adjacent numbers on the row below, write a program in a language of your choice, to find the maximum total from top to bottom in triangle.txt: a text file containing a triangle with 100 rows.

Solution:
Language: C#.
Source code is Triangle.cs.
Executable is Triangle.exe.

Triangle.exe will print out the following line:
"Max adjacent path sum: {number}", where {number} is the max path sum.

The program assumes that the triangle.txt file is in the same directory as Triangle.exe.
